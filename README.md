The Weather CLI Application is a Python-based command-line tool that allows users to quickly check real-time weather information for major Indian cities. This project integrates with the OpenWeatherMap API to fetch accurate and up-to-date weather data.

The application provides a simple and interactive user experience by offering both menu-based city selection and manual input. Users can choose from predefined cities such as Pune, Mumbai, Delhi, and more, or enter any city name of their choice. If no input is provided, the application defaults to Pune, ensuring smooth usability.

Once a city is selected, the program retrieves and displays essential weather details in a clean, structured terminal format, including temperature (in Celsius), weather conditions, humidity levels, and wind speed (converted to km/h).

This project demonstrates practical implementation of API integration, user input handling, error management, and clean output formatting in Python. It is designed with scalability in mind and can be extended into GUI, web, or mobile applications.

🚀 Key Features
Interactive CLI with menu + manual input
Default city selection for better UX
Real-time weather data using API
Clean and readable terminal output
Error handling for invalid inputs or API issues
🛠️ Tech Stack
Python
Requests Library
REST API (OpenWeatherMap)
